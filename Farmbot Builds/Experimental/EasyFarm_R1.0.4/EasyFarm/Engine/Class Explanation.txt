Class Roles:
	Engine - Holds all the function to control the bot. 
		e.g. Start to start the bot

	GameState - A class to hold and obtain environment detail for the bot,
		so that it may make intelligent decisions

	Streams - Holds IObservable streams of character data refreshed every second.

	UserOptions - Holds the data inputted from the main form so that bot may follow
		a user's commands.

	Player - Holds the functions involved for using a player in his/her environment.
		e.g. Battle to defeat units and Rest to heal
